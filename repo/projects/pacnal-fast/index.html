pacnal-fast is a poorly named, as well as poorly implemented, combination of pacapt, apt-fast, and nala.

I've always preferred pacman's syntax styling, nala is the best looking package manager available, but, it does not appear to have parallel downloading. Or even if it does, it's not as fast as apt-fast, so I made them call the each other.

(I realize I could just use aria2c directly for the downloads but then I would have to figure out how to use aria2c, and develop a visual output)

Just use the command belows:

sudo mv ~/pacapt-fast /usr/bin/pacman

sudo mv ~/nalafast /usr/bin/apt-fast

sudo apt-get install ./nala_0.11.0_all.deb

or if the main is not compatible:

sudo apt-get install ./nala-legacy_0.11.0_amd64.deb
